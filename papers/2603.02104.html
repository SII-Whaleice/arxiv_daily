ACDC: Adaptive Curriculum Planning with Dynamic Contrastive Control for Goal-Conditioned Reinforcement Learning in Robotic Manipulation

arXiv: 2603.02104 作者: Hengyan Liu Team 日期: 2026-03-02

📝 详细解读

研究背景与动机

在机器人操作领域，目标条件强化学习（GCRL）提供了一种基于学习的规划方法，使智能体能够在同一环境中实现多个目标。后见经验回放（HER）通过将失败轨迹重新标记为替代目标的成功经验，显著提高了GCRL的样本效率。现有研究通常结合基于多样性或熵驱动的优先级采样机制来改进经验选择。然而，这些方法普遍缺乏在整个训练过程中显式引导智能体学习焦点的能力，特别是在使学习信号与预期课程保持一致方面。这导致智能体可能在次优或不相关的经验上浪费学习能力，在稀疏奖励环境中尤为严重。

本文针对现有方法缺乏对学习过程的系统性、自适应引导这一根本局限性，提出了一个新的分层规划范式ACDC。其核心思路是集成自适应课程（AC）规划与动态对比（DC）控制，前者根据智能体成功率动态平衡探索（多样性）与利用（质量）以制定课程计划，后者通过约束对比学习执行该计划，实现与当前课程重点一致的经验选择。

方法详解

ACDC框架是一个双层协同系统，旨在通过精心设计的学习轨迹引导智能体。

图1：ACDC框架概述。规划层：自适应课程（AC）规划通过结合多样性和质量指标，并经由自适应加权机制生成轨迹优先级分数F(τ)。控制层：动态对比（DC）控制利用这些分数，通过排序和配对模块构建正/负样本对，然后采用两阶段对比学习进行经验选择。第一阶段训练编码器网络从构建的样本对中学习轨迹表示，第二阶段通过与智能体-编码器交互计算L2范数分数，以动态指导经验选择。

1. 自适应课程（AC）规划
该模块通过动态平衡从探索到利用的过渡来渐进式安排学习任务，构建一个性能感知的课程。其核心是为每个轨迹τ计算一个综合分数F(τ) = \tilde{d}_τ + λ(s_r, t) · \tilde{q}_τ，其中\tilde{d}_τ是归一化的多样性得分，\tilde{q}_τ是归一化的质量得分，λ(s_r, t)是自适应权重。

  • 多样性得分：使用行列式点过程（DPP）来度量轨迹在目标空间中的多样性。将轨迹划分为重叠的部分轨迹，计算每个部分轨迹的DPP行列式值，并求和得到总多样性得分d_τ，最后在回放缓冲区ℬ内进行最小-最大归一化得到\tilde{d}_τ。
  • 质量得分：衡量轨迹最终达到的目标g_T^{ac}与期望目标g的接近程度，计算公式为\tilde{q}_τ = exp(-(||g_T^{ac} - g||_2)^2 / (2σ^2))，其中σ=0.2。得分在0到1之间，越接近1表示轨迹质量越高。

图3：质量得分衡量轨迹最终达到的目标与期望目标的接近程度。该得分使用高斯函数将欧氏距离转换为0到1之间的值，从而平滑地区分成功与不成功的轨迹。

  • 自适应加权机制：权重λ(s_r, t) = λ_0 · (1 + η(s_r))^t根据当前成功率s_r和训练步数t动态调整。其中η(s_r)是一个基于成功率的自适应增长率：当s_r < θ_low时（挣扎阶段），η(s_r) = η_base · 0.5，鼓励更长时间的探索；当θ_low ≤ s_r ≤ θ_high时（学习阶段），η(s_r) = η_base，提供平衡的探索-利用；当s_r > θ_high时（掌握阶段），η(s_r) = η_base · 2.0，加速转向质量优先的利用。为了防止成功率在阈值附近波动导致模式切换不稳定，算法采用了指数移动平均（EMA）来平滑η(s_r)的更新。

2. 动态对比（DC）控制
该模块在操作层面执行AC的课程计划，通过学习的轨迹表示使智能体能够识别并优先处理与其当前学习阶段最相关的经验。

  • 动态对比配对：利用AC模块生成的综合分数F(τ)，将回放缓冲区中分数最高的前τ_p比例轨迹作为正样本集𝒫，分数最低的前τ_n比例轨迹作为负样本集𝒩。
  • 第一阶段：编码器网络训练：编码器采用融合设计，整合轨迹动态与课程上下文。它从轨迹中提取五个关键帧的达成目标作为输入，通过LSTM网络提取轨迹特征h_traj，同时将当前课程阶段λ(s_r, t)嵌入为h_λ。两者融合后生成联合编码z_{τ,λ(s_r,t)}。训练采用带有显式范数约束的对比损失：ℒ_total = ℒ_con + αℒ_norm。ℒ_con是一个自锚定的InfoNCE损失，防止正样本间的特征坍塌；ℒ_norm = max(0, μ_𝒩 - μ_𝒫 + m)则强制正样本编码的平均L2范数μ_𝒫至少比负样本的平均范数μ_𝒩高出边界m。

图5：训练后的编码器网络建立了一个双层表示架构。正轨迹占据优质区域（绿色），而负轨迹被限制在劣质区域（红色），点的大小表示L2范数的大小。这种结构实现了类别分离和基于范数的重要性排序。

  • 第二阶段：经验选择：使用训练好的编码器处理回放缓冲区中的所有轨迹，计算其联合编码的L2范数‖z_{τ,λ(s_r,t)}‖2。轨迹被采样的概率与其编码的范数成正比，即P_contrastive(τ) ∝ ‖z{τ,λ(s_r,t)}‖_2。这实现了从AC的高层课程目标到具体经验选择的转化。

创新点：与现有方法相比，ACDC的创新在于系统性地将自适应课程规划与对比学习相结合。AC模块不是固定课程，而是根据智能体实时性能（成功率）动态调整探索与利用的平衡；DC模块则利用对比学习，将AC的课程计划编码为轨迹表示空间的几何结构（区域分离与范数大小），从而实现与课程阶段严格对齐的、非启发式的经验选择。

实验与结果

实验设置：在OpenAI Gym的六个机器人操作任务上评估ACDC，涵盖两个机器人平台：Fetch环境（7自由度机械臂）和Shadow Dexterous Hand环境（24自由度灵巧手）。训练遵循标准HER框架，使用10个并行worker收集经验。

对比方法：与多种先进的基于HER的经验回放基线进行对比，包括：原始HER、CHER（课程引导的HER）、DTGSH（基于DPP的多样性采样）、FAHER（基于聚类的焦点感知HER）。

关键结果：

图8：在六个任务上的成功率对比曲线。ACDC（红色实线）在所有任务上均一致且显著地优于所有基线方法，在最终成功率和样本效率（达到特定成功率所需的训练周期数）方面都表现出色。

实验结果表明，ACDC在所有六个挑战性操作任务上均一致优于最先进的基线方法。例如，在最终任务成功率上，ACDC相比最佳基线平均提升超过15%。在样本效率方面，ACDC达到相同成功率所需的训练周期数显著减少。

消融实验分析：

图9：对AC和DC组件进行消融的实验结果。移除任一组件的性能均显著下降，验证了二者协同工作的必要性。仅使用AC（蓝色虚线）或仅使用DC（绿色点线）均无法达到完整ACDC（红色实线）的性能。

消融研究证实了AC和DC两个核心组件的必要性。移除任一组件的性能均显著下降。进一步分析表明：

  1. AC模块的作用：自适应权重机制是关键。固定权重（即固定探索-利用平衡）的策略性能不如自适应权重策略。
  2. DC模块的作用：范数约束损失ℒ_norm至关重要。没有该约束，编码器无法在表示空间中建立清晰的范数-重要性关联，导致经验选择效果下降。
  3. 阈值敏感性：论文对成功率阈值θ_low和θ_high进行了敏感性分析（参见论文图12），表明在合理范围内性能稳定，最优阈值因任务复杂度而异（Fetch任务设为0.3和0.65，Hand任务设为0.2和0.5）。

总结与启发

核心贡献：

  1. 提出了一个新颖的分层学习范式ACDC，首次系统地将自适应课程规划与动态对比控制相结合，用于GCRL中的经验选择。
  2. 设计了自适应课程（AC）规划器，能根据智能体实时成功率动态调整探索（多样性）与利用（质量）的平衡，实现无需环境模型的课程生成。
  3. 设计了动态对比（DC）控制器，通过带有范数约束的对比学习，将课程计划编码到轨迹表示中，实现与课程阶段严格对齐的、非启发式的经验选择。

局限性：论文提到，尽管在基准任务上表现优异，但该方法在更复杂、高维的任务（如仅使用像素输入的操作）上的泛化能力仍有待进一步验证。此外，自适应权重函数中的超参数（如增长率η_base和阈值θ）可能需要针对新领域进行微调。

研究启示：ACDC框架为GCRL中的经验复用提供了新思路。其“规划-控制”分层结构可扩展至其他需要渐进式学习的RL领域。动态对比控制中将抽象课程计划转化为具体表示空间几何结构的思想，为连接高层学习目标与低层策略优化提供了可借鉴的桥梁。未来工作可探索将ACDC应用于多任务学习、元学习，或结合世界模型进行更长期的课程规划。

💡 一句话总结

本论文针对目标条件强化学习在机器人操作任务中，因经验选择策略单一导致性能不佳的问题，提出了ACDC框架。其核心方法包含自适应课程规划，以智能体成功率为依据动态平衡探索与利用；以及动态对比控制，通过范数约束对比学习实现课程引导下的经验选择。实验表明，该方法在多个挑战性任务上，在样本效率和最终成功率上均优于现有先进基线。
