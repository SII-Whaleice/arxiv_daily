Mean-Flow based One-Step Vision-Language-Action

arXiv: 2603.01469 作者: Bin Zhao Team 日期: 2026-03-02

📝 详细解读

研究背景与动机

当前，基于流匹配（FlowMatching）的视觉-语言-动作（VLA）框架在生成高频动作块，特别是用于高度灵巧的机器人操作任务方面展现出显著优势。然而，其实际应用受到生成延迟过长的限制，这源于其固有的迭代采样需求和架构限制。具体而言，流匹配方法通过求解确定性常微分方程（ODE）来学习瞬时向量场，推理时需要通过欧拉积分等数值方法进行多步迭代来重建轨迹。当减少推理步数时，每个欧拉步覆盖的时间跨度变大，会放大偏差，导致生成的样本向数据集均值漂移，从而在实时效率与动作生成精度之间面临固有权衡。

本文针对这一关键瓶颈，提出了一种基于平均流（Mean-Flow）的一步VLA方法。其核心思路是摒弃对瞬时向量场的学习，转而学习一个平均去噪向量场，该场直接建模从噪声端点到数据端点的整体映射，从而避免了多步迭代去噪推理的需要，在单步或少数几步内即可生成高质量连续动作，显著提升了生成效率。

方法详解

整体框架由一个预训练的视觉语言模型（VLM）和一个Mean-Flow动作专家构成。VLM负责处理多模态输入（多视角RGB图像、语言指令、机器人本体感知状态），并输出统一的特征表示。Mean-Flow动作专家则基于此特征，预测平均去噪向量场，用于直接从噪声生成动作轨迹。

图1：Mean-Flow based One-Step VLA框架总览。预训练的VLM处理多模态输入。在训练期间，Mean-Flow动作专家近似于以VLM特征为条件的平均去噪向量场，并在推理时预测该平均向量场以获得动作A0。

该方法的核心创新在于用平均向量场（Mean Vector Field）替代传统流匹配中的瞬时向量场（Instantaneous Vector Field）。给定两个时间步r和t（0 ≤ r < t ≤ 1），平均向量场u定义为瞬时向量场v在时间区间[r, t]上的积分平均：u(zt, r, t) ≜ (1/(t-r)) ∫_r^t v(z_τ, τ) dτ，其中zt = at x + bt e是在数据样本x和噪声样本e之间的插值。

关键的技术细节在于推导出的“MeanFlow Identity”：u(zt, r, t) = v(zt, t) - (t-r) * du/dt。该恒等式揭示了平均向量场u与瞬时向量场v及u自身时间导数du/dt之间的关系。du/dt可通过雅可比向量积（JVP）高效计算。网络uθ被参数化以满足此恒等式。

训练时，时间步对(r, t)被随机采样。损失函数旨在最小化网络预测的平均向量场uθ与通过MeanFlow Identity计算得到的目标场utgt之间的差异：ℒ(θ) = 𝔼 || uθ(zt, r, t) - sg(utgt) ||₂²，其中sg(·)是停止梯度算子，用于防止梯度通过JVP项反向传播，避免高阶梯度计算带来的不稳定。论文发现，使用标准的L2损失训练效果不佳，因此采用了自适应损失函数：ℒ = (1/B) Σ [1/(||Δ_i||² + c)^(1-γ) * ||Δ_i||²]，其中γ是控制自适应程度的超参数。

在推理时，动作生成策略极为高效。对于单步生成，直接从噪声A1 ∼ 𝒩(0, I)通过一步计算得到动作：A0 = A1 - uθ(A1, 0, 1)。若采用多步生成，则遵循公式：A_{γ-δ} = A_γ - δ * uθ(A_γ|O_γ)，其中δ为步长。

与现有方法相比，其创新点具体体现在：1) 原理创新：用学习跨时间区间的平均向量场替代学习瞬时向量场，从根本上改变了生成过程的建模方式。2) 效率突破：得益于平均向量场直接建模全局映射的特性，理论上可实现一步高质量生成，无需依赖多步数值积分。3) 训练简化：方法基于确定的平均场目标进行训练，无需额外的蒸馏或一致性约束，训练流程简单稳定。

实验与结果

实验在自制的SO-101六自由度机械臂上进行，使用两个RGB相机（腕部视角和俯视视角）。评估了三个真实的机器人操作任务：拾放（pick-place）、堆叠（stacking）和分拣（sorting）。每个任务收集了100条演示数据。

对比的基线方法包括最新的FlowMatching-based VLA方法SmolVLA和Diffusion-based方法Diffusion Policy。实验平台为NVIDIA RTX 4090（训练）和搭载Intel Core i5-10200H与NVIDIA GTX 1650的笔记本电脑（在线推理）。

图2：真实机器人实验设置及三个操作任务示意图。展示了SO-101机械臂以及拾放、堆叠和分拣任务场景。

关键实验结果如下：

  1. 生成速度：所提出的Mean-Flow based One-Step VLA的生成速度比SmolVLA快8.7倍，比Diffusion Policy快83.9倍。
  2. 任务成功率：在参数敏感性研究中，在NFE（函数评估次数）=5的设置下，模型在拾放任务上取得了最高84.5%的成功率（流比率=0.2时）。使用自适应损失（γ=0.5）进一步将成功率提升至86%。

图3：不同流比率（flow-ratio）下的模型性能。流比率控制着训练中r=t（瞬时场）样本的比例。结果表明，完全训练在平均场（比率=1.0）或包含适量瞬时场样本（比率=0.2, 0.5）对性能有显著影响，最佳平衡点在0.2。

图4：不同自适应损失超参数γ下的成功率（流比率=0.2，NFE=5）。γ=0.5时取得最佳性能（86%），而纯L2损失（γ=1.0）性能严重下降至9.5%，证明了自适应损失对于稳定训练的重要性。

图5：与基线方法SmolVLA和Diffusion Policy的推理时间对比。Mean-Flow方法在单步（NFE=1）和5步（NFE=5）生成模式下均具有显著的速度优势。

消融实验与参数分析总结：

  • 流比率（Flow-Ratio）：这是一个关键设计选择。完全训练在平均向量场（比率=1.0）会导致成功率暴跌至4.5%。引入20%的瞬时向量场样本（比率=0.2）能提供局部精度监督，与平均场学习形成互补，取得了最佳成功率（84.5%），平衡了局部控制与全局行为。
  • 损失函数：使用自适应损失（γ=0.5）而非标准L2损失，对于稳定优化和提升最终精度至关重要，因为它能抑制长间隔样本带来的高方差目标的影响。
  • 函数评估次数（NFE）：NFE从1增加到5时，任务成功率显著上升；继续增加到10仅带来边际改善。这表明NFE=5在生成质量与计算成本之间达到了最佳平衡。NFE=1时性能较低，因为一步预测跨越整个时间区间[0,1]的全局映射对于复杂、多模态的VLA数据来说过于困难。

总结与启发

本文的核心贡献在于：1) 提出了首个将MeanFlow生成方法集成到VLA框架中的工作，即Mean-Flow based One-Step VLA，用于机器人操作。2) 通过用平均去噪向量场预测器替代传统的FlowMatching机制，实现了显著的效率提升，生成速度比先进方法快8.7倍。3) 通过真实机器人实验验证了该框架在单步和多步生成模式下的鲁棒性能。

论文自身提到的局限性在于，尽管单步生成速度极快，但在NFE=1时生成的轨迹精度较低。为了取得高成功率，仍需使用适度的NFE（如5步），这反映了在模型拟合误差与求解器积分误差之间进行权衡的必要性。

本研究对后续工作的启示包括：MeanFlow为构建高效、实时的VLA策略提供了一个有前景的新方向。其“一步生成”的潜力尤其适用于对延迟要求极高的实时控制场景。此外，该方法无需蒸馏或复杂调度即可实现高效生成，简化了训练流程，为在资源受限设备上部署大型VLA模型提供了可能。未来的研究可以探索如何进一步优化平均向量场的学习，以提升单步生成的精度，或将该框架扩展到更复杂的多任务、长时程规划场景中。

💡 一句话总结

本文针对基于流匹配的视觉-语言-动作模型存在生成延迟高的问题，提出一种**均值流单步生成方法**。该方法通过**解决噪声诱导问题**，消除了传统流匹配的一致性约束，从而实现**高效的单步动作生成**。真实机器人实验表明，其生成速度比SmolVLA快**8.7倍**，比Diffusion Policy快**83.9倍**，显著提升了基于VLA的机器人操作效率。
